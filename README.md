# SQL Practice Repository

## Overview

This repository contains SQL examples covering fundamental and intermediate database concepts. It demonstrates practical SQL skills including data retrieval, joins, subqueries, aggregations, and window functions.

## Repository Structure

- Database_Schema.sql – Database schema and sample data
- Basic_Queries.sql – Fundamental SQL queries
- Joins.sql – INNER JOIN and LEFT JOIN examples
- Subqueries.sql – Nested query examples
- Window_Functions.sql – Analytical SQL functions
